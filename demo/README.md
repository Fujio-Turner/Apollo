# Spring Gardens — Flower Demo

Welcome to the **Spring Gardens** demo project! This little knowledge base
documents the flowers we grow at Spring Gardens, tended by our head
gardener **Maria Alvarez**.

## Flowers in this collection

- [Roses](roses.md) — our signature climbing and bush varieties.
- [Tulips](tulips.md) — the spring stars that open the season.
- [Sunflowers](sunflowers.txt) — the giants that close it.
- [Care Guide](care_guide.md) — watering, pruning, and pest tips that
  apply to roses, tulips, sunflowers, and daisies alike.

## About Spring Gardens

Spring Gardens is a four-acre demonstration garden in the foothills.
Maria Alvarez has run the gardens since 2019 and specializes in
companion planting: roses with garlic, tulips with daffodils,
sunflowers with squash, and daisies along every border.
